# Strong Password Generator

A simple and efficient Strong Password Generator that creates secure, random passwords using uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Generates strong, unpredictable passwords
- Customizable length
- Includes letters, numbers, and symbols
- Lightweight and easy to use

## How to Use
1. Run the script
2. Enter the desired password length
3. Get your secure, randomly generated password
